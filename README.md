# Growvix - Leading Global Multiple Solutions Provider

Growvix is a leading global multiple solutions provider that helps businesses and individuals achieve their goals through web development, mobile apps, SaaS, SEO, and digital marketing.

## Features

- **Web Development**: Custom websites and web applications
- **Mobile App Development**: iOS and Android applications
- **SaaS Products**: Scalable software as a service solutions
- **SEO & Content Strategy**: Search engine optimization and content marketing
- **Facebook Ads**: Targeted social media advertising
- **Brand Strategy**: Comprehensive branding and marketing strategies

## Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS
- **Deployment**: Netlify
- **Forms**: Netlify Forms

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
```

## Deployment

This project is configured for deployment on Netlify. Push to your repository and connect it to Netlify for automatic deployments.

## Project Structure

```

├── thank-you/           # Contact form thank you page

```
